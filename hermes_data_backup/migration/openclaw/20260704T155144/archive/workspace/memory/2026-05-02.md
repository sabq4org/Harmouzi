## 🔧 نظام الواتساب وعزل السياقات

### ما تم:
- تغيير `dmPolicy` من `pairing` إلى `allowlist` لقناة الواتساب
- تفعيل `crossContext.allowAcrossProviders = true` مع marker دائم
- سلطان المالكي وصلته رسالة الاعتذار

### الدروس المُسجّلة (مهمة جداً):
- **الخطأ الكلاسيكي:** أبو محمد أرسل "أنا سلطان" من تيليجرامه → صدّقت النص وتجاهلت chat_id
- **القاعدة الحديدية:** chat_id + channel = مصدر الحقيقة الوحيد
- **`[[reply_to_current]]` لا يعمل عبر القنوات** — يجب استخدام `message` tool صراحة
- **الإعداد الحالي:** `allowAcrossProviders: true` مع marker = حل دائم بدون فتح/قفل

### ربط الأصدقاء بالـ chat_id:
- أبو محمد تيليجرام: `chat_id: 1079517049` → دائماً أبو محمد
